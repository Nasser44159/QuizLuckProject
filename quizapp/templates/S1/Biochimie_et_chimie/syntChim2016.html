Chimie 2016

1.) Parmi les propositions suivantes, cochez la (les) proposition(s) exacte(s) :

  • A- La matière est tout ce qui possède une masse et qui occupe l’espace.
  • B- L’unité de masse atomique (u.m.a.) est le douzième (1/12) de la masse d’un atome de carbone 12C.
  • C- La formation d’un noyau à partir de protons et de neutrons donne lieu à une libération d’énergie considérable correspondant à une perte de masse Δm.
  • D- L’énergie d’ionisation de l’atome d’hydrogène est égale à +13,64 eV.
  • E- Le Li+ est un ion hydrogénoïde.

2.) Quelle(s) est (sont) la(les) réponse(s) correcte(s) ?

  • A- (n = 0 ; l = 0 ; ml = 0) correspond à l’orbitale 1s.
  • B- (n = 1 ; l = 0 ; ml = 0) correspond à l’orbitale 2s.
  • C- (n = 1 ; l = 1 ; ml = 0) correspond à l’orbitale 2p.
  • D- (n = 3 ; l = 0 ; ml = 0) correspond à l’orbitale 3d.
  • E- (n = 3 ; l = 1 ; ml = 0) correspond à l’orbitale 3p.

3.) Au sujet de l’atome :

  • A- Le nombre de masse d’un élément est égal à (Z + N) ; avec Z = Numéro atomique et N = nombre de neutrons.
  • B- La masse d’un proton et celle d’un neutron sont strictement les mêmes.
  • C- Les charges du proton et du neutron sont opposées.
  • D- Pour l’hydrogène, le rapport entre la taille d’un atome et celle de son noyau est de l’ordre 100.
  • E- L’électron est utilisé pour son numéro de masse A.
